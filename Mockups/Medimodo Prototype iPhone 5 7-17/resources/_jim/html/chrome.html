Google Chrome Advise

Due to security restrictions Google Chrome limits the access to local files. Follow these instructions to allow access:
  • Windows: Close all running instances of Google Chrome and
    • OPTION 1: Open Justinmind Prototyper and go to File\Preferences. Click on 'Browse' in 'Default browser' and select Chrome from your file system (C:\Users\[Username]\AppData\Local\Google\Chrome).
    • OPTION 2: Edit Chrome shortcut with "PATH_TO_CHROME\chrome.exe" --allow-file-access-from-files
  • Mac: Go to Google Chrome main menu, select 'Quit' and
    • OPTION 1: Open Justinmind Prototyper and go to Prototyper Pro\Preferences. Click on 'Browse' in 'Default browser' and select Chrome from your file system (Applications\Google Chrome).
    • OPTION 2: Open a Terminal and type: open -a "Google Chrome" --args --allow-file-access-from-files
